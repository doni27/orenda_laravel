# API documentation:

The following is just a simple list of the api end points:

> GET /api/users
> GET /api/users/:id
> POST /api/add-users

> GET /api/product
> GET /api/product/:id
> POST /api/add-produk

> GET /api/koli/common
> GET /api/koli/common/:id
> POST /api/add-koli
> GET /api/update-koli/:id
